3.8 / 4.3 ScoreCalculationMethod Method used to calculate score. Strings are predefined, with the following values:Average - Calculates the score from the average score of each post in the topic made by a user. Max - Calculates the score based on the maximum score of any one post in the topic for a user.Min - Calculates the score based on the minimum score of any one post in the topic for a user.Mode Max - Given the number of posts, selects the score that is repeated the most. If there is a tie, it takes the highest score.Mode Min - Given the number of posts, selects the score that is repeated the most. If there is a tie, it takes the lowest score.Sum - Calculates the score from the sum of scores on each post in a topic for a user.Blank value (null) - The score is not calculated based on the scores of each post (Manual updates). varchar 19
